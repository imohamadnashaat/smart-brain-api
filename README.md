# Smart Brain API

## Getting Started

1. Clone this repo
2. Run `npm install`
3. Run `npm start`
4. You must add your own API key in the `.env` file to connect to Clarifai API
5. Add your own database credentials to `.env`

You can grab Clarifai API key [here](https://www.clarifai.com/)

** Make sure you use postgreSQL instead of mySQL for this code base.
